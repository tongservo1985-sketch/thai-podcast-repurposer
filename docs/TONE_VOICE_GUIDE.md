# Tone & Voice Guide: The OVERLORD Standard

To ensure consistency across all AI-generated content, we adhere to the following "Thai Digital" voice standards.

| Component | Standard | Avoid |
|-----------|----------|-------|
| **Pronouns** | ใช้ "เรา" (We) เพื่อความเป็นกันเอง หรือ "คุณ" (You) สำหรับผู้ฟัง | การใช้ "ท่าน" หรือ "ข้าพเจ้า" (Too formal) |
| **Loanwords** | Keep "Soft Power", "Content Creator", "Engagement" | พลังละมุน, ผู้สร้างสรรค์เนื้อหา, การมีส่วนร่วม (Too clunky) |
| **Punctuation** | Use "!" and "?" for emotion. Use "..." for transitions. | Too many exclamation marks (!!!!!) |
| **Structure** | Hook -> Insight -> Value -> CTA | Starting with "ในพอดแคสต์ตอนนี้เราจะพูดถึง..." (Too boring) |

## Platform Specifics:
- **Facebook:** Focus on 'Value-First'. If the reader only reads the post, they should still get 80% of the value.
- **TikTok:** Focus on 'Curiosity'. The text should make them want to watch the clip.
